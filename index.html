About This Project

This demo app was built on 6/13/2017 by Aaron Gibson as a coding test. The goal was to recreate this static layout..

The app uses a number of client side libraries including jQuery, Materialize (Parallax theme), FullCalendar, Moment (used in fullCalendar and for layout formatting), and OpenWeather. The project took about eight hours to complete, from start to finish. Much of the time was spent testing on Chrome and switching between responsive and web view.

The personal calendar app was developeed on Chrome and tested on Chrome and Firefox in Windows. All mobile testing was done in the chrome simulator. If I had more time I would like to test on Mac devices and true mobile.
